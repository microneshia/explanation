演習問題(6)：重ね合わせの原理とポテンシャルの導入

この演習問題は、講義で学んだ「重ね合わせの原理」と「ポテンシャルの導入」に関する理解度を測るものです。以下に各問題の詳しい解答と解説を示します。

1. 問題1：線形性の確認

演算子 \(L\) が線形であるためには、斉次性 \(L(aX) = aL(X)\) と加法性 \(L(X+Y) = L(X) + L(Y)\) を満たす必要があります。

1.1. (1) \(L(X) = 3X + \frac{dX}{dt}\)

この演算子は斉次性と加法性を両方満たすため、線形です。

1.2. (2) \(L(X) = X\frac{dX}{dt}\)

\(L(aX) = a^2 X\frac{dX}{dt} \neq aL(X)\) となるため、斉次性を満たさず、非線形です。

1.3. (3) \(L(X) = \sin(X)\)

一般に \(\sin(aX) \neq a\sin(X)\) であるため、斉次性を満たさず、非線形です。

2. 問題2：重ね合わせの原理

全電場 \(\boldsymbol{E}\) の表現:
電荷密度が \(\rho_1 + \rho_2\) の場合、全電場は \(\boldsymbol{E} = \boldsymbol{E}_1 + \boldsymbol{E}_2\) と表されます。これは重ね合わせの原理に基づきます。

原理の根拠:
マクスウェル方程式が線形な方程式であるためです。例えばガウスの法則 \(\nabla \cdot \boldsymbol{E} = \rho / \varepsilon_0\) は、\(\boldsymbol{E}\) と \(\rho\) に対して線形です。

電場のエネルギー密度について:
全エネルギー密度 \(u\) は、\(u = u_1 + u_2 + \varepsilon_0 \boldsymbol{E}_1 \cdot \boldsymbol{E}_2\) となります。相互作用項が加わるため、エネルギー密度については重ね合わせの原理は成り立ちません。

3. 問題3：ベクトルの恒等式の証明

各恒等式は、成分計算によって証明できます。

3.1. (1) \({\boldsymbol \nabla}\times({\boldsymbol \nabla}\phi)={\boldsymbol 0}\)

スカラー関数の2階偏微分の順序が交換可能であることから、各成分がゼロとなり、恒等式が成り立ちます。

3.2. (2) \({\boldsymbol \nabla}\cdot({\boldsymbol \nabla}\times{\boldsymbol A})={\boldsymbol 0}\)

こちらも同様に、2階偏微分の順序交換可能性により、各項が打ち消し合ってゼロとなり、恒等式が成り立ちます。

3.3. (3) \({\boldsymbol \nabla}\times({\boldsymbol \nabla}\times {\boldsymbol A})={\boldsymbol \nabla}({\boldsymbol \nabla}\cdot{\boldsymbol A})-{\boldsymbol \nabla}^2{\boldsymbol A}\)

左辺のx成分を計算し、\(\frac{\partial^2 A_x}{\partial x^2}\) を足し引きすることで、右辺のx成分と一致することを示せます。y, z成分も同様です。

4. 問題4：ゲージ変換

ゲージ変換後のポテンシャル \(\phi' = \phi - \frac{\partial \chi}{\partial t}\) と \(\boldsymbol{A}' = \boldsymbol{A} + \boldsymbol{\nabla} \chi\) を用いて、新しい電磁場 \(\boldsymbol{E'}\), \(\boldsymbol{B'}\) を計算します。

磁場 \(\boldsymbol{B'}\) の計算:

$$ \boldsymbol{B'} = \boldsymbol{\nabla} \times \boldsymbol{A'} = \boldsymbol{\nabla} \times \boldsymbol{A} + \boldsymbol{\nabla} \times (\boldsymbol{\nabla} \chi) = \boldsymbol{B} + \boldsymbol{0} = \boldsymbol{B} $$

電場 \(\boldsymbol{E'}\) の計算:

$$ \boldsymbol{E'} = -\boldsymbol{\nabla} \phi' - \frac{\partial \boldsymbol{A'}}{\partial t} = \left(-\boldsymbol{\nabla}\phi - \frac{\partial \boldsymbol{A}}{\partial t}\right) + \boldsymbol{\nabla}\left(\frac{\partial \chi}{\partial t}\right) - \frac{\partial}{\partial t}(\boldsymbol{\nabla}\chi) = \boldsymbol{E} $$

結論: \(\boldsymbol{E'} = \boldsymbol{E}\), \(\boldsymbol{B'} = \boldsymbol{B}\) となり、電場と磁場はゲージ変換に対して不変です。

まとめ
