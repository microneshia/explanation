電機システムII 宿題7 詳細解説

この記事では、電機システムIIの宿題7で出題された三相誘導電動機に関する問題について、詳細な解説を行います。 各問題の条件整理、基本的な公式の適用、計算過程、そして最終的な解答をステップバイステップで説明します。 この記事を読むことで、誘導電動機の特性計算（すべり、トルク、効率、抵抗制御など）についての理解を深めることができます。

問題1. 巻線形誘導機の最大トルクと始動時抵抗

問題文: 4極、定格出力20kW、200V、50Hzの巻線形回転子を持つ三相誘導機があり、二次巻線には抵抗を外付け出来るようになっている。二次巻線を短絡した場合この誘導機は1380rpmの時に最大トルクを発生する。以下の問に答えよ。

与えられた情報:

  • 極数 p = 4
  • 定格出力 P_out = 20 kW
  • 定格電圧 V = 200 V
  • 周波数 f = 50 Hz
  • 二次巻線短絡時、最大トルク発生時の回転数 N_m = 1380 rpm
  • (1-2)で与えられる回転子の一相あたりの巻線抵抗 R_2 = 0.40 Ω

(1-1) 二次巻線短絡時に最大トルクを発生するすべりを求めよ。

解説:
同期回転数 \(N_s\) は電源周波数 \(f\) と極数 \(p\) で決まります。すべり \(s\) は、同期回転数 \(N_s\) と回転子の回転数 \(N\) から次のように定義されます。

$$ s = \frac{N_s - N}{N_s} $$

最大トルクを発生するときの回転数が与えられているので、このときのすべり \(s_m\) を計算します。

式と計算:

  1. 同期回転数 \(N_s\) の計算:
    \[ N_s = \frac{120 \times f}{p} \] \[ N_s = \frac{120 \times 50 \text{ Hz}}{4} = \frac{6000}{4} = 1500 \text{ rpm} \]
  2. 最大トルク発生時のすべり \(s_m\) の計算:
    \[ s_m = \frac{N_s - N_m}{N_s} \] \[ s_m = \frac{1500 \text{ rpm} - 1380 \text{ rpm}}{1500 \text{ rpm}} = \frac{120}{1500} = 0.080 \]

解答: 最大トルクを発生するすべり \(s_m = 0.080\)

(1-2) 始動(s=1)の時に最大トルクを出すためには一相あたりの二次巻線の外付け抵抗を何Ωにすればよいか。ただし回転子の一相あたりの巻線抵抗は0.40Ωであるとする。

解説:
誘導電動機の最大トルクを発生するすべり \(s_m\) は、二次回路の全抵抗 \(R_{2,total}\) と二次回路の漏れリアクタンス \(X_{20}\) (始動時、すなわち \(s=1\) での回転子周波数におけるリアクタンス) の比で決まります。この関係は「比例推移」として知られ、二次抵抗 \(R_{2,total}\) を変えると、最大トルクの値自体は変わらずに、最大トルクを発生するすべり \(s_m\) が \(R_{2,total}\) に比例して移動します。

$$ s_m = \frac{R_{2,total}}{X_{20}} $$

式と計算:

  1. 二次回路の漏れリアクタンス \(X_{20}\) の算出:

    条件1: 二次巻線短絡時 (外付け抵抗 \(R_{ext1} = 0\))。回転子固有抵抗 \(R_2 = 0.40 \, \Omega\)。このときの全二次抵抗 \(R_{2,total1} = R_2 = 0.40 \, \Omega\)。最大トルク時すべり \(s_{m1} = 0.080\)。

    \[ s_{m1} = \frac{R_{2,total1}}{X_{20}} \] \[ \implies X_{20} = \frac{R_{2,total1}}{s_{m1}} \] \[ X_{20} = \frac{0.40 \, \Omega}{0.080} = 5.0 \, \Omega \]
  2. 始動時 (\(s=1\)) に最大トルクを出すための全二次抵抗 \(R_{2,total2}\) の計算:

    条件2: 始動時に最大トルクを発生させたいので、このときの最大トルク時すべり \(s_{m2} = 1\)。必要な回転子全抵抗を \(R_{2,total2} = R_2 + R_{ext2}\) とします (\(R_{ext2}\) は求める外付け抵抗)。

    \[ s_{m2} = \frac{R_{2,total2}}{X_{20}} \] \[ \implies R_{2,total2} = s_{m2} \times X_{20} \] \[ R_{2,total2} = 1 \times 5.0 \, \Omega = 5.0 \, \Omega \]
  3. 必要な外付け抵抗 \(R_{ext2}\) の計算:
    \[ R_{2,total2} = R_2 + R_{ext2} \] \[ R_{ext2} = R_{2,total2} - R_2 \] \[ = 5.0 \, \Omega - 0.40 \, \Omega = 4.6 \, \Omega \]

解答: 始動時に最大トルクを出すために必要な外付け抵抗は 4.6 Ω

問題2. 回転数比例負荷時の抵抗制御 (トルクはすべりに比例)

問題文: 巻線形回転子を持つ三相誘導機があり、二次巻線の抵抗\(R_2\)を自由に変更することが出来る。この誘導機を用いて回転数に比例する負荷を運転させた時すべりが1%であった。この回転子の抵抗を増加させてすべりを2%に変更するには二次巻線の抵抗\(R_2\)を何倍にすれば良いか。ただし誘導機の発生するトルクはすべりに比例するとする。

与えられた情報:

  • 負荷トルク \(T_L \propto N\) (回転数 \(N\) に比例)
  • 誘導機の発生トルク \(T_M \propto s\) (すべり \(s\) に比例)
  • 初期状態: すべり \(s_1 = 1\% = 0.01\)
  • 変更後状態: すべり \(s_2 = 2\% = 0.02\)

(2-1) すべり 1%およびすべり 2%で回転している時の負荷トルクをそれぞれ \(T_1\)、\(T_2\)とする。負荷トルクが回転子の回転数に比例することから \(T_2\)を \(T_1\)で表せ。

解説:
回転数 \(N = N_s (1-s)\) であり、負荷トルク \(T_L = k_L N\) です。

式と計算:

  1. 状態1 (すべり \(s_1 = 0.01\)):
    \[ N_1 = N_s (1 - s_1) = N_s (1 - 0.01) = 0.99 N_s \] \[ T_1 = k_L N_1 = k_L \times 0.99 N_s \]
  2. 状態2 (すべり \(s_2 = 0.02\)):
    \[ N_2 = N_s (1 - s_2) = N_s (1 - 0.02) = 0.98 N_s \] \[ T_2 = k_L N_2 = k_L \times 0.98 N_s \]
  3. \(T_2\) と \(T_1\) の比:
    \[ \frac{T_2}{T_1} = \frac{k_L \times 0.98 N_s}{k_L \times 0.99 N_s} = \frac{0.98}{0.99} \] \[ T_2 = \frac{0.98}{0.99} T_1 \] \[ T_2 \approx 0.99 T_1 \]

解答: \(T_2 = 0.99 T_1\)

(2-2) 回転子の抵抗を増加させずにトルク \(T_2\)を発生させるために必要なすべり \(s_0\)を求めよ。

解説:
元の二次抵抗 \(R_{2,orig}\) のままでトルク \(T_2\) を発生させる場合を考えます。誘導機の発生トルクはすべりに比例する (\(T_M = k_M s\)) と仮定します。ここで \(k_M\) は元の抵抗値での比例定数です。

式と計算:

  • 状態1 (\(T_1, s_1=0.01\)): \(T_1 = k_M s_1 = k_M \times 0.01\)
  • 状態0 (\(T_2, s_0\)): \(T_2 = k_M s_0\)
\[ \frac{T_2}{T_1} = \frac{k_M s_0}{k_M \times 0.01} = \frac{s_0}{0.01} \]

(2-1) の結果 \(T_2/T_1 = 0.99\) を用いると、

\[ 0.99 = \frac{s_0}{0.01} \] \[ s_0 = 0.01 \times 0.99 = 0.0099 \]

解答: \(s_0 = 0.0099\)

(2-3) 回転子の抵抗を増加させて2%で運転するには回転子の巻線の抵抗を何倍にする必要があるか (ヒント:比例推移)

解説:
同じトルク \(T_2\) を発生させるのに、

  • 元の抵抗 \(R_{2,orig}\) では、すべり \(s_0 = 0.0099\) が必要。
  • 新しい抵抗 \(R_{2,new}\) では、すべり \(s_2 = 0.02\) で運転したい。
比例推移の原理より、低すべり領域で特定のトルクを発生させるには、\( \frac{R_{2,new}}{R_{2,orig}} = \frac{s_2}{s_0} \) という関係が成り立ちます。

式と計算:

\[ \frac{R_{2,new}}{R_{2,orig}} = \frac{s_2}{s_0} \] \[ \frac{R_{2,new}}{R_{2,orig}} = \frac{0.02}{0.0099} \approx 2.020202... \] \[ \frac{R_{2,new}}{R_{2,orig}} = 2.02 \]

解答: 回転子の巻線抵抗を 2.02 倍 にする必要がある。

問題3. 回転数二乗比例負荷時の抵抗制御

問題文: 巻線形回転子を持つ三相誘導機があり、二次巻線の抵抗\(R_2\)を自由に変更することが出来る。この誘導機を用いて回転数の二乗に比例する負荷を運転させた時すべりが1%であった。この回転子の抵抗を増加させてすべりを2%に変更するには二次巻線の抵抗\(R_2\)を何倍にすれば良いか。ただし誘導機の発生するトルクはすべりに比例するとする。

与えられた情報:

  • 負荷トルク \(T_L \propto N^2\) (回転数 \(N\) の二乗に比例)
  • 誘導機の発生トルク \(T_M \propto s\) (すべり \(s\) に比例)
  • 初期状態: すべり \(s_1 = 1\% = 0.01\)
  • 変更後状態: すべり \(s_2 = 2\% = 0.02\)

解説:
問題2と同様のアプローチですが、負荷トルクの特性が異なります。平衡点では \(T_L = T_M\)。

式と計算:

  1. 負荷トルクと回転数の関係:
    \[ N = N_s (1-s) \] \[ T_L = k_L N^2 = k_L (N_s (1-s))^2 \]
  2. 平衡点でのトルク \(T_{L1}, T_{L2}\) の関係:

    状態1 (\(s_1 = 0.01\)): \(N_1 = N_s(1-0.01) = 0.99 N_s\)

    \[ T_{L1} = k_L (0.99 N_s)^2 \]

    状態2 (\(s_2 = 0.02\)): \(N_2 = N_s(1-0.02) = 0.98 N_s\)

    \[ T_{L2} = k_L (0.98 N_s)^2 \]

    トルクの比:

    \[ \frac{T_{L2}}{T_{L1}} = \frac{k_L (0.98 N_s)^2}{k_L (0.99 N_s)^2} = \left(\frac{0.98}{0.99}\right)^2 \]
  3. 元の抵抗 \(R_{2,orig}\) のままでトルク \(T_{L2}\) を発生させるためのすべり \(s_0\):

    誘導機発生トルク \(T_M \propto s\) (抵抗一定なら)。平衡点では \(T_M = T_L\)。

    \(T_{M1} = k_M s_1\) であり \(T_{M1} = T_{L1}\)。

    \(T_{M0} = k_M s_0\) であり \(T_{M0} = T_{L2}\)。

    \[ \frac{T_{M0}}{T_{M1}} = \frac{T_{L2}}{T_{L1}} = \frac{s_0}{s_1} \] \[ s_0 = s_1 \times \frac{T_{L2}}{T_{L1}} = 0.01 \times \left(\frac{0.98}{0.99}\right)^2 \] \[ s_0 = 0.01 \times \frac{0.9604}{0.9801} \approx 0.009799 \]
  4. 比例推移の適用 (同じトルク \(T_{L2}\) を発生させる):

    同じトルクを発生させる場合、すべり \(s\) は二次抵抗 \(R_2\) に比例すると解釈 (\(R_{2,new}/R_{2,orig} = s_2/s_0\))。

    \[ \frac{R_{2,new}}{R_{2,orig}} = \frac{s_2}{s_0} \] \[ = \frac{0.02}{0.01 \times \left(\frac{0.98}{0.99}\right)^2} \] \[ = \frac{0.02}{0.01} \times \left(\frac{0.99}{0.98}\right)^2 \] \[ = 2 \times \frac{0.9801}{0.9604} \approx 2 \times 1.0205122 \approx 2.04102 \] \[ \frac{R_{2,new}}{R_{2,orig}} = 2.04 \]

解答: 回転子の巻線抵抗を 2.04 倍 にする必要がある。

問題4. 定格負荷運転時の特性計算

問題文: 5kW、4極の三相誘導機がある。この誘導機を60Hzの電源に接続して定格負荷運転(出力5kW)を行うと回転数は1728rpmであった。この誘導機について以下の問に答えよ。ただし、無負荷時(s=0)の損失は200W、一次側に換算した時の巻線抵抗値\(R_1\)、\(R_2'\)は \(R_1:R_2'=1:2\)であったとする。

与えられた情報:

  • 定格出力 \(P_{out} = 5 \text{ kW} = 5000 \text{ W}\)
  • 極数 \(p = 4\)
  • 周波数 \(f = 60 \text{ Hz}\)
  • 定格負荷時の回転数 \(N_r = 1728 \text{ rpm}\)
  • 無負荷損 \(P_{noload} = 200 \text{ W}\) (鉄損 \(P_{fe}\) と機械損 \(P_{fw}\) の合計)
  • 一次抵抗 \(R_1\) と一次換算二次抵抗 \(R_2'\) の比 \(R_1 : R_2' = 1 : 2\)

(4-1) 同期速度は何 rpm か

式と計算:

\[ N_s = \frac{120 \times f}{p} \] \[ = \frac{120 \times 60 \text{ Hz}}{4 \text{ 極}} = \frac{7200}{4} = 1800 \text{ rpm} \]

解答: 同期速度 \(N_s = \) 1800 rpm

(4-2) 定格負荷時のすべりを求めよ

式と計算:

\[ s = \frac{N_s - N_r}{N_s} \] \[ = \frac{1800 \text{ rpm} - 1728 \text{ rpm}}{1800 \text{ rpm}} = \frac{72}{1800} = \frac{1}{25} = 0.040 \]

解答: 定格負荷時のすべり \(s = \) 0.040

(4-3) 定格負荷時の効率は何%か

解説:
効率 \(\eta = P_{out} / P_{in}\)。入力 \(P_{in} = P_{out} + \text{損失合計}\)。 損失 = 一次銅損 \(P_{c1}\) + 二次銅損 \(P_{c2}'\) + 無負荷損 \(P_{noload}\)。 二次銅損 \(P_{c2}' = P_{out} \frac{s}{1-s}\) (ここで \(P_{out}\) を \(P_m\) と近似して計算)。

式と計算:

  1. 二次銅損 \(P_{c2}'\):
    \begin{align*} P_{c2}' &= P_{out} \times \frac{s}{1-s} \\ &= 5000 \text{ W} \times \frac{0.040}{1-0.040} \\ &= 5000 \times \frac{0.040}{0.960} = \frac{200}{0.960} \text{ W} \\ &\approx 208 \text{ W} \end{align*}
  2. 一次銅損 \(P_{c1}\):

    一次電流 \(I_1\) と一次換算二次電流 \(I_2'\) がほぼ等しいと仮定すると、\(P_{c1} / P_{c2}' \approx R_1 / R_2' = 1/2\)。

    \[ P_{c1} = \frac{P_{c2}'}{2} = \frac{208 \text{ W}}{2} = 104 \text{ W} \]
  3. 一次入力 \(P_{in}\):
    \[ P_{in} = P_{out} + P_{c1} + P_{c2}' + P_{noload} \] \[ P_{in} = 5000 \text{ W} + 104 \text{ W} + 208 \text{ W} + 200 \text{ W} = 5512 \text{ W} \]
  4. 効率 \(\eta\):
    \[ \eta = \frac{P_{out}}{P_{in}} = \frac{5000 \text{ W}}{5512 \text{ W}} \approx 0.907111... \] \[ \eta = 90.7 \% \]

解答: 定格負荷時の効率 \(\eta = \) 90.7 %

(4-4) 全負荷時のトルクは何 Nm か

解説:
出力トルク \(T_{out} = P_{out} / \omega_r\)、ここで \(\omega_r\) は回転子の角速度。

式と計算:

  1. 回転子の角速度 \(\omega_r\):
    \[ N_r = 1728 \text{ rpm} = \frac{1728}{60} \text{ rps} = 28.8 \text{ rps} \] \[ \omega_r = 2\pi N_{r,rps} = 2\pi \times 28.8 \text{ rad/s} \]
  2. 出力トルク \(T_{out}\):
    \begin{align*} T_{out} &= \frac{P_{out}}{\omega_r} \\ &= \frac{5000 \text{ W}}{2\pi \times 28.8 \text{ rad/s}} \\ &\approx 27.631... \text{ Nm} \\ T_{out} &= 27.6 \text{ Nm} \end{align*}

    （別解として、ギャップパワー \(P_{ag} = P_{out} + P_{c2}' = 5000 + 208 = 5208 \text{ W}\) を用い、同期角速度 \(\omega_s = 2\pi N_s / 60 = 2\pi \times 1800 / 60 = 60\pi \text{ rad/s}\) から \(T_{ag} = P_{ag}/\omega_s = 5208 / (60\pi) \approx 27.62 \text{ Nm}\) としてもほぼ同じ値が得られます。）

解答: 全負荷時のトルク \(T_{out} = \) 27.6 Nm

問題5. かご形誘導機の拘束試験

問題文: 定格出力40kW、6極、440V、50Hzのあるかご形三相誘導機の拘束試験を行った結果は以下の通りである。ただし鉄損は無視できるとする。線間電圧60V、線電流60A、三相入力電力3.2kW、端子間の直流抵抗0.212Ω。

与えられた情報 (拘束試験時, s=1):

  • 印加線間電圧 \(V_{L,br} = 60 \text{ V}\)
  • 線電流 \(I_{L,br} = 60 \text{ A}\)
  • 三相入力電力 \(P_{3ph,br} = 3.2 \text{ kW} = 3200 \text{ W}\)
  • 固定子巻線の端子間直流抵抗 \(R_{dc,term} = 0.212 \, \Omega\) (Y結線と仮定し、2相分の抵抗)
  • 鉄損無視
  • 一般情報: 定格出力 \(40 \text{ kW}\), \(p=6\)極, 定格電圧 \(V_{L,rated} = 440 \text{ V}\), \(f=50 \text{ Hz}\)

(5-1) 一相あたりの一次抵抗値を求めよ。

解説:
Y結線の場合、端子間の直流抵抗は2相分の抵抗となります。\(R_{dc,term} = 2 \times R_{1,dc}\)。交流における実効抵抗 \(R_{1,ac}\) は皮相効果などにより直流抵抗より若干大きくなることがありますが、ここでは直流抵抗値を用います。

式と計算:

\[ R_1 = R_{1,dc} = \frac{R_{dc,term}}{2} \] \[ = \frac{0.212 \, \Omega}{2} = 0.106 \, \Omega \]

解答: 一相あたりの一次抵抗値 \(R_1 = \) 0.106 Ω

(5-2) 拘束試験時の二次入力 \(P_2\)を求めよ。

解説:
二次入力 \(P_2\) はギャップパワー \(P_{ag}\) に等しいです。拘束試験時 (\(s=1\)) の入力電力 \(P_{3ph,br}\) は、一次銅損 \(P_{c1,br}\) と二次入力 \(P_{ag,br}\) (これは全て二次銅損 \(P_{c2}',br\) になる) の合計です（鉄損は無視）。 \(P_{3ph,br} = P_{c1,br} + P_{ag,br}\)。 Y結線なので、相電流 \(I_{1,ph,br} = I_{L,br} = 60 \text{ A}\)。

式と計算:

  1. 一次銅損 \(P_{c1,br}\):
    \[ P_{c1,br} = 3 \times I_{1,ph,br}^2 \times R_1 \] \[ = 3 \times (60 \text{ A})^2 \times 0.106 \, \Omega \] \[ = 3 \times 3600 \times 0.106 = 10800 \times 0.106 = 1144.8 \text{ W} \]
  2. 二次入力 \(P_{ag,br}\) (または \(P_2\)):
    \[ P_{ag,br} = P_{3ph,br} - P_{c1,br} \] \[ = 3200 \text{ W} - 1144.8 \text{ W} = 2055.2 \text{ W} \] \[ P_{ag,br} = 2055 \text{ W} \]

解答: 拘束試験時の二次入力 \(P_2 = \) 2055 W

(5-3) 発生トルクを求めよ

解説:
拘束試験時 (\(s=1\)) の発生トルク \(T_{br}\) は、二次入力 \(P_{ag,br}\) を同期角速度 \(\omega_s\) で割ったものです。 \(\omega_s = 2\pi N_{s,rps} = \frac{4\pi f}{p}\)。

式と計算:

  1. 同期角速度 \(\omega_s\):
    \[ N_s = \frac{120 \times f}{p} = \frac{120 \times 50 \text{ Hz}}{6 \text{ 極}} = 1000 \text{ rpm} \] \[ \omega_s = \frac{2\pi N_s}{60} = \frac{2\pi \times 1000}{60} = \frac{100\pi}{3} \text{ rad/s} \approx 104.719... \text{ rad/s} \]
  2. 発生トルク \(T_{br}\):
    \begin{align*} T_{br} &= \frac{P_{ag,br}}{\omega_s} \\ &= \frac{2055 \text{ W}}{100\pi/3 \text{ rad/s}} \\ &\approx 19.623... \text{ Nm} \\ T_{br} &= 19.6 \text{ Nm} \end{align*}

解答: 発生トルク \(T_{br} = \) 19.6 Nm

(5-4) この電動機を拘束したまま、定格電圧を印加した。二次入力を求めよ

解説:
拘束状態 (\(s=1\)) で電圧を変えた場合、インピーダンスはほぼ変わらないと仮定できます。電力は電圧の二乗に比例します (\(P \propto V^2\))。 二次入力 \(P'_{ag}\) は、拘束試験時の二次入力 \(P_{ag,br}\) を基準に、電圧比の二乗をかけます。

式と計算:

\begin{align*} P'_{ag} &= P_{ag,br} \times \left(\frac{V_{L,rated}}{V_{L,br}}\right)^2 \\ &= 2055 \text{ W} \times \left(\frac{440 \text{ V}}{60 \text{ V}}\right)^2 \\ &= 2055 \times \left(\frac{44}{6}\right)^2 = 2055 \times \left(\frac{22}{3}\right)^2 \\ &= 2055 \times \frac{484}{9} \\ &= 2055 \times 53.777... = 110500 \text{ W} \\ P'_{ag} &= 110.5 \text{ kW} \end{align*}

解答: 定格電圧印加時の二次入力 \(P'_{ag} = \) 110.5 kW

(5-5) 発生トルクを求めよ。

解説:
トルクは二次入力 (ギャップパワー) に比例し、同期角速度は変わりません。 \(T' = P'_{ag} / \omega_s\)。

式と計算:

\begin{align*} T' &= \frac{P'_{ag}}{\omega_s} \\ &= \frac{110500 \text{ W}}{100\pi/3 \text{ rad/s}} \\ &\approx 1055.19... \text{ Nm} \\ T' &= 1055 \text{ Nm} \end{align*}

解答: 発生トルク \(T' = \) 1055 Nm

問題6. 誘導機の逆相制動運転

問題文: 三相誘導機を逆相制動運転させ誘導制動機として用いている。電動機の電源周波数を60Hzとする。今重量W[kg]の重量物を吊り下げて一定速度v[m/s]で巻き降ろしたとき、誘導機の回転数は400rpmであった。以下の問に答えよ。なお重力加速度はg[m/s^2]、誘導機の電源周波数は60Hz、極数12、一次の巻線抵抗、鉄損を無視する。

与えられた情報:

  • 電源周波数 \(f = 60 \text{ Hz}\)
  • 重量物の質量 \(M = W \text{ [kg]}\) (問題文のWを質量Mとします)
  • 巻き降ろし速度 \(v \text{ [m/s]}\)
  • 逆相制動時の回転数 \(N_{r,brake} = 400 \text{ rpm}\) (通常の回転方向と逆)
  • 重力加速度 \(g \text{ [m/s}^2\text{]}\)
  • 極数 \(p = 12\)
  • 一次巻線抵抗 \(R_1\) 無視、鉄損 \(P_{fe}\) 無視

(6-1) 重量物を一定速度で巻き降ろすのに必要な力は何Nか

解説:
一定速度で巻き降ろすということは、重量物にかかる重力と、誘導機が発生する（支持する）力が釣り合っている状態です。誘導機は制動しているので、上向きの力を発生させており、その力の大きさは重力に等しいです。

式と計算:

$$ F = M \times g = Wg \text{ [N]} $$

解答: 必要な力 \(F = \) Wg [N]

(6-2) 三相誘導モータが吸収している機械エネルギーは何Wか

解説:
機械的パワー \(P_m = \text{力} F \times \text{速度} v\)。このパワーは、重量物が失う位置エネルギーの割合であり、誘導機の軸に入力される機械的パワーとなります。誘導機が「吸収している」と表現されているので、正の値で答えます。

式と計算:

\[ P_{m,absorbed} = F \times v \] \[ = (Wg) \times v = Wgv \text{ [W]} \]

解答: 誘導モータが吸収している機械エネルギー \(P_{m,absorbed} = \) Wgv [W]

(6-3) 誘導機のすべりを求めよ

解説:
同期速度 \(N_s = 120f/p\)。逆相制動なので、回転磁界の方向に対して回転子は逆方向に回転しています。回転磁界の回転数を \(+N_s\) とすると、回転子の回転数は \(-N_{r,brake}\) となります。 すべり \(s = (N_s - N_{rotor,actual}) / N_s\)。

式と計算:

  1. 同期速度 \(N_s\):
    \[ N_s = \frac{120 \times f}{p} \] \[ = \frac{120 \times 60 \text{ Hz}}{12 \text{ 極}} = \frac{7200}{12} = 600 \text{ rpm} \]
  2. すべり \(s\):

    \(N_{rotor,actual} = -N_{r,brake} = -400 \text{ rpm}\)

    \[ s = \frac{N_s - (-N_{r,brake})}{N_s} = \frac{N_s + N_{r,brake}}{N_s} \] \[ s = \frac{600 \text{ rpm} + 400 \text{ rpm}}{600 \text{ rpm}} \] \[ = \frac{1000}{600} = \frac{10}{6} = \frac{5}{3} \approx 1.6666... \] \[ s = 1.67 \]

解答: 誘導機のすべり \(s = \) 1.67

(6-4) 一次側電源が供給している電力を求めよ。(ヒント:一次巻線、鉄損を無視するので二次入力に等しい)

解説:
一次巻線抵抗と鉄損を無視するので、一次入力 \(P_1\) は二次入力 \(P_{ag}\) (ギャップパワー) に等しいです (\(P_1 = P_{ag}\))。 誘導機が吸収する機械的パワー \(P_{m,absorbed}\) は、電動機から見ると負の機械的出力 \(-P_{m,developed}\) となります。 つまり \(P_{m,developed} = -P_{m,absorbed} = -Wgv\)。 また、\(P_{m,developed} = P_{ag} (1-s)\)。

式と計算:

\begin{align*} -Wgv &= P_{ag} (1-s) \\ -Wgv &= P_{ag} \left(1 - \frac{5}{3}\right) \\ -Wgv &= P_{ag} \left(-\frac{2}{3}\right) \\ P_{ag} &= \frac{-Wgv}{-2/3} = \frac{3}{2} Wgv = 1.5 Wgv \end{align*}

一次入力 \(P_1 = P_{ag}\)。

解答: 一次側電源が供給している電力 \(P_1 = \) 1.5 Wgv [W]

(6-5) 二次銅損を求めよ

解説:
二次銅損 \(P_{c2}' = P_{ag} \times s\)。 または、エネルギー保存則から、二次銅損は、二次入力(ギャップパワー)と機械的出力の差: \(P_{c2}' = P_{ag} - P_{m,developed} = P_{ag} - (-P_{m,absorbed}) = P_{ag} + P_{m,absorbed}\)。

式と計算:

使用する値: \(P_{ag} = 1.5 Wgv\), \(s = 5/3\), \(P_{m,absorbed} = Wgv\)。

方法1: \(P_{c2}' = P_{ag} \times s\)

\begin{align*} P_{c2}' &= (1.5 Wgv) \times \frac{5}{3} \\ &= \left(\frac{3}{2} Wgv\right) \times \frac{5}{3} \\ &= \frac{5}{2} Wgv = 2.5 Wgv \end{align*}

解答: 二次銅損 \(P_{c2}' = \) 2.5 Wgv [W]

まとめ

本解説では、三相誘導電動機に関する様々な計算問題に取り組みました。すべりの定義、トルク特性、効率計算、抵抗制御、拘束試験、逆相制動といった重要な概念について、具体的な数値例を通して理解を深めることができたかと思います。誘導電動機の動作原理と特性を把握することは、電機システムの設計や運用において非常に重要です。
