Input Leap 設定ガイド

MacBookのキーボードとマウスで、隣のWindows PCをラクラク操作！

はじめに：目標と準備

このガイドで出来るようになること

MacBookのキーボードとトラックパッドを使い、マウスカーソルをMacの画面からWindowsの画面へシームレスに移動させて、そのままWindows PCを操作できるようになります。

準備するもの

  • MacBook (macOS)
  • Windows デスクトップPC
  • 共通のネットワーク接続（例：両方のPCが同じ家庭用Wi-Fiルーターに接続されている状態）

全体像：サーバーとクライアント

2台のPCの役割を決めます。このように覚えると簡単です。

  • サーバー（親機）：命令を出すPC。キーボードやマウスがつながっている方です。→ 今回はMacBook
  • クライアント（子機）：命令を受け取るPC。操作される方です。→ 今回はWindows PC

ステップ1：Input Leapのダウンロードとインストール

まず、MacとWindowsの両方のパソコンにInput Leapをインストールします。ここでは、インストーラーが提供されているv3.0.2を例に解説します。

  1. Input Leapの公式リリース一覧ページを開きます。
    Input Leap 公式リリース・ページ
  2. v3.0.2のリリースを探し、お使いのPCに合ったファイルをダウンロードします。

    ページを少しスクロールして「v3.0.2」の項目を見つけ、その下の「Assets」という部分をクリックしてファイル一覧を表示します。

    【Windowsの方】
    ファイル一覧の中から、InputLeap_3.0.2_windows_qt6.exe というファイルをダウンロードしてください。これがWindows用のインストーラーです。
    【Macの方】どちらの.dmgファイルを選べばいいの？
    Mac用には2種類のファイルがあります。お使いのMacの心臓部である「チップ（CPU）」の種類によって、ダウンロードするファイルが異なります。以下の方法で確認しましょう。
    1. 画面左上の Appleのリンゴマーク () をクリックします。
    2. メニューから「このMacについて」を選択します。
    3. 開いたウィンドウに表示されている項目を確認します。
      • 「チップ」という項目に「Apple M1」や「Apple M2」のように書かれている場合 → あなたのMacは Apple Silicon です。
      • 「プロセッサ」という項目に「Intel Core i5」のように書かれている場合 → あなたのMacは Intel です。

    確認できたら、対応するファイルをダウンロードしてください。

    • Apple Silicon の場合 → InputLeap_3.0.2_macos_AppleSilicon.dmg
    • Intel の場合 → InputLeap_3.0.2_macos_x86_64.dmg
  3. それぞれのパソコンでインストールします。

    【Macでのインストール】
    ダウンロードした.dmgファイルを開き、出てきた「Input Leap」アイコンを「Applications」フォルダにドラッグ＆ドロップします。

    【Windowsでのインストール】
    ダウンロードした.exeファイルをダブルクリックし、画面の指示に従って「Next」や「Install」を押して進めれば完了です。

ステップ2：サーバーの設定 (MacBook側)

  1. Input Leapを起動し、「Server」を選びます。
    「アプリケーション」フォルダからInput Leapを起動し、「Server」を選択します。
  2. IPアドレスをメモします。
    メイン画面の IP addresses: に表示されているIPアドレス（例：192.168.1.5）を、正確にメモしてください。後でWindows側で使います。
  3. サーバー設定画面を開き、PCを配置します。
    「Configure Server...」をクリックし、設定画面右上のPCアイコンをマス目の中にドラッグ＆ドロップします。Windows PCが机の上にある物理的な位置に合わせると直感的です。
  4. クライアント（Windows PC）の名前を設定します。
    今置いたPCアイコン（"Unnamed"）をダブルクリックし、「Screen name」の欄に、後でWindows側で確認する「スクリーン名」と全く同じ名前を入力します。（例: Windows-PC）
  5. 【最重要】Macのセキュリティ設定を許可します。
    • Input Leapのメイン画面で「Start」ボタンを押します。
    • 「アクセシビリティ」に関する警告が出たら、「"システム設定"を開く」をクリックします。
    • 「プライバシーとセキュリティ」＞「アクセシビリティ」の画面で、「Input Leap」のスイッチをオンにします。（パスワードを求められます）
    • 設定後、一度Input Leapを「Stop」し、再度「Start」を押して「running」状態にしておきましょう。

ステップ3：クライアントの設定 (Windows PC側)

  1. Input Leapを起動し、「Client」を選びます。
    「Client」にチェックが入っていることを確認します。
  2. スクリーン名を確認し、Mac側の設定と合わせます。
    画面上部の Screen name: を確認します。この名前が、Mac側で設定した名前と完全に一致しているか確認してください。違っていたらMac側を修正します。
  3. サーバーのIPアドレスを入力します。
    「Server IP」の空欄に、メモしたMacBookのIPアドレスを正確に入力します。
  4. Windowsファイアウォールを許可します。
    「Start」ボタンを押した際に「Windows Defender ファイアウォール」の警告が出たら、「プライベート ネットワーク」にチェックを入れて「アクセスを許可する」をクリックします。

ステップ4：接続確認とトラブルシューティング

接続完了！

両方のPCで「Start」を押しInput Leap is runningと表示されれば準備完了です。Macのマウスカーソルを画面の端に持っていき、Windows側に移動すれば成功です！

うまくいかない場合のチェックリスト

あわてずに、ひとつずつ確認してみてください！
  1. IPアドレスは正しいですか？ (MacのIPをWindowsに正確に入力しましたか？)
  2. スクリーン名は完全に一致していますか？ (大文字・小文字・ハイフンもすべて同じですか？)
  3. Macの「アクセシビリティ」許可は有効ですか？ (これが一番多い原因です。再度確認を！)
  4. 同じネットワークに接続していますか？ (両方とも同じWi-Fiですか？)
  5. アプリを正しい順番で再起動しましたか？ (両方Stop → ①サーバー(Mac)をStart → ②クライアント(Win)をStart の順で試してください。)
  6. キーボードのScroll Lockが有効になっていませんか？ (Scroll Lockがオンになっていると、画面間の移動がロックされることがあります。キーボードのScroll Lockランプが消えているか確認してください。)
  7. PCを再起動してみましたか？ (最終手段ですが、効果的な場合も多いです。)
